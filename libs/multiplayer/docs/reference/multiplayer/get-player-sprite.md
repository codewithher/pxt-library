# get Player Sprite

Get the sprite assign to a player.

```sig
mp.getPlayerSprite(mp.playerSelector(mp.PlayerNumber.One))
```

## Parameters

* **player**: the player whose sprite will be returned.

## Returns

* a [sprite](/reference/types/sprite) that is assigned to the **player**.
